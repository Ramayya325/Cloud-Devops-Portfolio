## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/your-username/your-repo.git
```

Navigate to project folder:

```
cd your-repo
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
